# 钢琴块 (Piano Block)

一个为星露谷物语1.6制作的mod，添加可配置的音乐方块，让你在游戏中创作音乐！

## 功能特性

### 🎵 核心功能
- **可配置音符** - 每个方块可以配置多个音符序列
- **12个半音** - 支持C、C#、D、D#、E、F、F#、G、G#、A、A#、B
- **8个八度** - 从低音到高音，音域广阔
- **可调曲速** - 100ms到2000ms可调节
- **直观界面** - 简单易用的配置菜单

### 🎹 使用方法
1. **获取钢琴块**
   - 在罗宾的木匠铺购买（需要农业等级3，500金币）
   - 或者学习配方自己制作（需要：10个木材 + 1个铜锭）

2. **放置方块**
   - 像其他物品一样放置在地图上

3. **配置音乐**
   - 右键点击方块打开配置菜单
   - 使用 **↑/↓** 按钮调整音高（半音）
   - 使用 **8↑/8↓** 按钮调整八度
   - 点击 **添加** 按钮添加更多音符
   - 点击 **X** 删除不需要的音符
   - 调整曲速控制播放速度

4. **播放音乐**
   - 在配置菜单中点击 **播放** 按钮测试
   - 保存后，每次右键点击都会播放配置的音符序列

### 🎼 创作提示
- 每个方块可以是一个音符、和弦或短旋律
- 将多个方块组合起来创作复杂的音乐
- 使用不同的曲速制造节奏变化
- 尝试不同的八度组合获得丰富的音色

## 安装要求

- Stardew Valley 1.6或更高版本
- [SMAPI](https://smapi.io/) 4.0.0或更高版本
- [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915) 2.0.0或更高版本

## 安装方法

1. 安装SMAPI和Content Patcher
2. 下载本mod的最新版本
3. 解压到 `Mods` 文件夹
4. 确保同时安装了主mod和 `[CP] Piano Block` 文件夹
5. 启动游戏
